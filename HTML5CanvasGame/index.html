Silly Ball


A weird version of "Breakout"



Move the paddle left and right with the mouse or press left and right keys on the keyboard.
The goal is to break all 20 bricks with the silly ball.
(Be careful though, the ball will speed up with each brick hit.)


This game I made with the help of MDN's online tutorial - "2d breakout game using pure javascript". I changed some things myself and added new code to make it a little more unique.
