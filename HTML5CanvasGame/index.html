Silly Ball


A weird version of "Breakout"



Use the mouse or Left/Right keys to control the paddle.
The goal is to break all of the bricks with the silly ball.
Break the 'Golden Brick' for a bonus.
Be careful though, the ball is more elusive than you think!


I made this game with the foundational knowledge from the online tutorial "2d breakout game using pure javascript" - by the Mozilla Developer Network. However, I've upgraded the gameplay to make it a bit more challenging! Hope you enjoy.
